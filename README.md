Apple Sales Analytics Dashboard | Power BI

Project Overview  
An interactive Power BI dashboard analyzing Apple’s global sales, category performance, store distribution, and repair status to support business insights and decision-making.

---

Key Features  
- ₹6.17B total sales analyzed across regions  
- Category-level insights: Tablet highest, Smart Speaker least purchased  
- Store summary: 75 stores, US leading  
- Repair analytics segmented by product categories  
- Multi-page dashboard with KPIs, charts, and DAX measures  

---

Tech Stack  
- Power BI  
- Power Query  
- DAX  
- Data Modeling
